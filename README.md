# Projeto Calculadora

## 1º Passo
Executar no terminal o comando: 
´´´
docker build . -t calculadora
´´´ 
na pasta do projeto

## 2º Passo

Executar no terminal o comando: 
´´´
docker run -t --rm  -p 8081:80 calculadora
´´´ 
na pasta do projeto

## 3º Passo
Acessar navegador na url: http://localhost:8081/

# Tecnologias utilizadas
*HTML
*CSS
*JavaScript
